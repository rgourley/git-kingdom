Privacy Policy

Last updated: March 2026

Git Kingdom is a free, open-source project. We take your privacy seriously and collect the minimum data needed to make the game work.

GitHub OAuth

When you sign in, we use GitHub OAuth to authenticate your identity. We request read-only access to your public profile — specifically your username, avatar URL, and list of public repositories.

We never request access to private repositories, and we never write anything to your GitHub account.

What We Store

When you sign in and claim your repos, we store the following in our database:

  • Your GitHub username and avatar URL
  • A list of your public repositories with 1+ stars (name, language, star count)
  • A Supabase authentication token (for session management)

We do not store your email address, GitHub access token, or any private repository information.

Cookies

Git Kingdom uses a session cookie set by Supabase Auth to keep you signed in. This cookie is strictly functional — we do not use advertising cookies, tracking pixels, or cross-site tracking of any kind.

Analytics

We may use basic analytics (such as Vercel Web Analytics) to understand page views and performance. These are anonymized and do not track individual users across sites.

Third-Party Services

  • GitHub API — to fetch public repository and contributor data
  • Supabase — database and authentication
  • Vercel — hosting and serverless functions
  • Google Fonts — for the Silkscreen pixel font

Data Deletion

If you want your data removed from Git Kingdom, you can sign out at any time. To request complete deletion of your stored data, open an issue on our GitHub repository or email us. We'll remove your record within 30 days.

Open Source

Git Kingdom is fully open source. You can inspect exactly what data we collect and how we handle it by reviewing the code on GitHub.
