About Git Kingdom

Explore GitHub as a Fantasy RPG World

Git Kingdom transforms the entire GitHub ecosystem into an interactive pixel-art fantasy world. Every public repository becomes a building, every programming language becomes a kingdom, and every contributor becomes a walking citizen in the city.

The RPG Metaphor

🏰

Repos = Buildings

The more stars a repo has, the bigger the building. A 1-star repo is a small cottage. React is a massive citadel.

⚔️

Languages = Kingdoms

Each programming language is its own kingdom — TypeScript, Python, Rust, Go — each with its own city and citizens.

👥

Contributors = Citizens

Top contributors to repos in each kingdom walk the streets as pixel-art citizens. The top contributor is the king.

★

Stars = Building Size

GitHub stars determine building rank — from hovels (21+ stars) to manors (5k+) to citadels (100k+).

Claim Your Repos

Sign in with GitHub to claim your public repositories. Any repo with at least 1 star becomes a building in its language's kingdom. You can find your buildings by clicking your avatar in the header.

Built With

Git Kingdom is built with Phaser 3 for the game engine, Vite for the frontend build, Supabase for the database and auth, and deployed on Vercel. The pixel art uses custom-drawn building templates rendered on HTML5 Canvas.

Open Source

Git Kingdom is open source. Check out the code on GitHub.
